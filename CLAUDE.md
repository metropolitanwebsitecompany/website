# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is the marketing website for Metropolitan, a web design company serving Toledo businesses. It is a single static HTML page deployed via GitHub Pages at `metropolitanwebsitecompany.com`.

## Structure

- `index.html` — the entire site: inline CSS, no JavaScript, no build step
- `CNAME` — configures the custom domain for GitHub Pages

## Deployment

Changes pushed to `main` are deployed automatically by GitHub Pages. There is no build process — edit `index.html` directly.
